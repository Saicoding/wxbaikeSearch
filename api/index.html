基本信息

彦希，1996年2月17日出生于山东省，中国内地男演员、歌手 [1]  。
2016年，出演个人首部电视剧《明星志愿》，从而开始演艺生涯 [2]  ；5月，加盟古装电视剧《轩辕剑之汉之云》，剧中饰演徒维 [3]  。2017年，主演都市励志爱情剧《逆袭之星途璀璨》，饰演暖男言楚非 [4]  ；同年，加盟偶像养成类真人秀节目《超次元偶像》，完成个人综艺首秀 [5]  。2018年，主演都市情感剧《大约是爱》，饰演卫卿 [6]  。2019年，参加《2019央视元宵晚会》，亮相晚会开场歌舞《喜气洋洋闹元宵》 [7]  ；同年，主演院线电影《小山河》 [8]  ；10月12日，参加青春体娱跨界类型节目《超新星全运会第二季》 [9]  ；10月27日，出演由胡储玺导演执导《亲爱的药王大人》饰演楚之墨 [10]  。

彦希演艺经历

编辑
2016年3月，彦希受邀出席乐视全球生态盛典，首次在国内公开亮相 [2]  。5月，在潘文杰执导的古装电视剧《轩辕剑之汉之云》中，彦希突破形象饰演外表高冷，内心柔软的徒维，这也是他出演的首部古装剧 [3]  ；12月，根据明星养成模拟经营游戏《明星志愿3》改编的明星养成类网络剧《明星志愿》在爱奇艺正式上线，彦希在剧中饰演了男二号天晴，同时他还为该剧献唱了插曲《My Everything》及《初恋》 [13]  。
2017年3月，出演于荣光执导，由吴磊、林允领衔主演的电视剧《斗破苍穹》，剧中饰演萧历 [11]  ；彦希与宋轶、push联合主演都市励志爱情剧《逆袭之星途璀璨》，在剧中饰演帅气温柔的人气明星言楚非 [4]  ；同年，彦希加盟偶像养成类真人秀节目《超次元偶像》，完成个人综艺首秀 [5]  。
2018年1月，搭档许晓诺共同主演许珮珊执导的都市情感剧《大约是爱》，剧中饰演能撩妹、爱毒舌的霸道总裁卫卿 [6]  。
2019年，与陈昊宇合作主演由彭臣导演并编剧的院线电影《小山河》，在片中饰演性格温暖体贴的男主角小龙 [8]  ；10月12日，参加青春体娱跨界类型节目《超新星全运会第二季》 [9]  ；10月27日，出演由胡储玺导演执导《亲爱的药王大人》饰演楚之墨 [10]  。
